## ⚠️ 关键注意事项

### 1. Bot Token 安全

- ❌ 不要提交到 Git 仓库
- ✅ 使用环境变量存储
- ✅ 定期轮换 Token

### 2. Webhook vs Long Polling

| 模式 | 适用场景 | 要求 |
|------|----------|------|
| Webhook | 生产环境 | HTTPS 域名 |
| Long Polling | 开发/测试 | 无特殊要求 |

### 3. Telegram API 限制

- 消息频率：30 条/秒
- callback_data：≤64 字节
- 消息长度：4096 字符
- 文件大小：2GB

### 4. 群组隐私模式

- 默认开启：Bot 无法读取普通消息
- 关闭方法：在 @BotFather 使用 `/setprivacy` 命令
